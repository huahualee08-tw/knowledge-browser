# 知識庫瀏覽器 (Knowledge Browser) - 產品規格書

## 1. 產品概述

**產品名稱**：知識庫瀏覽器 (Knowledge Browser)
**產品類型**：Web App (響應式網頁)
**核心功能**：讓滑滑能快速瀏覽、管理 Google Drive 知識庫中的文件
**目標使用者**：滑滑（東元電機設計經理，AI 愛好者）

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 優先級 |
|------|------|--------|
| **資料同步** | 從 Google Drive 同步知識庫檔案列表與內容 | P0 |
| **文件列表** | 顯示所有知識庫文件，支援分頁/無限滾動 | P0 |
| **智慧摘要** | AI 自動生成每個文件的簡短摘要 (50-100字) | P0 |
| **分類瀏覽** | 按分類資料夾 (AI資訊、程式開發、OpenClaw 等) 篩選 | P1 |
| **關鍵字搜尋** | 全文搜尋標題、內容、摘要 | P1 |
| **文件檢視** | 點擊後顯示完整文件內容 (Markdown 渲染) | P0 |
| **收藏功能** | 標記常用文件方便快速存取 | P2 |

### 2.2 資料來源

Google Drive 資料夾結構：
```
~/Google Drive/我的雲端硬碟/
├── AI資訊/
├── 程式開發/
├── OpenClaw/
├── 個人成長/
├── 投資理財/
├── YouTube_影片摘要/
└── 其他/
```

## 3. 技術架構

### 3.1 技術選型

| 層面 | 技術 | 理由 |
|------|------|------|
| **前端框架** | React + Vite | Vibe Coding 友好，開發快速 |
| **UI 庫** | Tailwind CSS + shadcn/ui | 美觀、快速客製化 |
| **狀態管理** | Zustand | 輕量、簡單 |
| **Google API** | Google Drive API v3 | 官方 API，穩定 |
| **AI 摘要** | MiniMax API (現有) | 滑滑已有 API |
| **部署** | Vercel / Cloudflare Pages | 免費、快速 |

### 3.2 系統架構圖

│  │  Sidebar │  │  Main    │  │  Preview │              │

│  └──────────┘  └──────────┘  └──────────┘              │
└──────────────────────┬──────────────────────────────────┘
                       │
        ┌──────────────┴──────────────┐
        │       Backend API           │

        └──────────────┬──────────────┘

    ┌──────────────────┼──────────────────┐

### 3.3 資料流程

```
1. 使用者登入 (Google OAuth)
2. App 呼叫 Google Drive API 取得檔案列表
3. 對於新檔案，呼叫 MiniMax API 生成摘要
4. 摘要存入本地快取 (localStorage/IndexedDB)
5. 使用者瀏覽/搜尋時從快取讀取
6. 定期背景同步更新
```

- **字體**：Inter (正文) + JetBrains Mono (代碼)

| 螢幕寬度 | 布局 |

## 5. 功能詳述

### 5.1 文件卡片

```
┌─────────────────────────────────────┐
│ 📄 OpenClaw API 免費來源整理         │
│                                       │
│ 摘要：本文整理了各種免費 AI API      │
│ 來源，包括 Groq、NVIDIA NIM 等...   │
│                                       │
│ 🏷️ AI資訊   📅 2026-03-09   ⭐    │
└─────────────────────────────────────┘
```

### 5.2 搜尋功能

- 支援即時搜尋 (輸入後立即過濾)
- 搜尋範圍：標題、摘要、內容
- 熱門搜尋建議
- 搜尋結果高亮

### 5.3 摘要生成

- 首次同步時生成摘要
- 支援批量處理 (每次 10 篇)
- 摘要長度：50-100 字
- 儲存格式：
```json
{
  "fileId": "xxx",
  "title": "OpenClaw API 整理",
  "summary": "本文整理了各種免費 AI API 來源...",
  "generatedAt": "2026-03-10T23:00:00Z"
}
```

## 7. 風險與對策

| 風險 | 對策 |
|------|------|
| Google API 配額限制 | 實作快取機制，減少 API 呼叫 |
| 摘要生成費時 | 背景非同步處理，顯示載入狀態 |
| 文件過多載入慢 | 分頁 + 虛擬滾動 |
| OAuth 認證複雜 | 使用 Firebase Auth 簡化 |
